🎲 Dice Roller – Утилита для броска костей 
Простая и мощная утилита для симуляции бросков игральных костей.
Поддерживает произвольное количество кубиков, граней и бросков, выводит статистику и красочные результаты.
Реализована на 7 языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Бросок костей – от 1 до 100 кубиков за раз.

Настраиваемые грани – классический d6, d20, d100 и любые другие.

Многократные броски – серия бросков с выводом статистики.

Режимы вывода:

--sum – показать только сумму.

--stats – показать мин, макс, среднее, медиану.

--verbose – вывести каждый бросок в цвете.

Цветной вывод – каждый кубик выделяется цветом, а критические значения подсвечиваются.

Сохранение истории – все броски можно сохранить в файл (опционально).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [количество_кубиков] [количество_граней] [опции]
Аргументы
количество_кубиков – число кубиков (по умолчанию 1).

количество_граней – число граней (по умолчанию 6).

Опции
Опция	Описание
-r, --rolls <N>	Количество бросков (по умолчанию 1)
-s, --sum	Вывести только сумму броска
-t, --stats	Показать статистику (мин, макс, среднее, медиана)
-v, --verbose	Подробный вывод (каждый кубик в цвете)
-o, --output <file>	Сохранить результат в файл
-h, --help	Справка
Примеры
bash
# Бросок одного d6
python dice.py

# Бросок 3d20
python dice.py 3 20

# Серия из 10 бросков 2d6 со статистикой
python dice.py 2 6 -r 10 -t

# Подробный вывод 5d10
python dice.py 5 10 -v

# Сохранить результат в файл
python dice.py 4 8 -r 5 -o results.txt

🧠 Логика работы
Парсинг аргументов – определение количества кубиков, граней, опций.

Генерация бросков – для каждого броска создаётся список случайных чисел от 1 до числа граней.

Вывод – в зависимости от опций показывается подробный или суммарный результат.

Статистика – вычисляются минимальное, максимальное, среднее и медианное значения.

Цвет – каждый кубик отображается в одном из цветов радуги, критические значения (1 и максимум) выделяются ярко.

✨ Дополнительные фичи
Поддержка больших чисел – работа с произвольной точностью (где возможно).

Имитация реальных костей – равномерное распределение.

Экспорт в файл – сохранение результатов для дальнейшего анализа.
